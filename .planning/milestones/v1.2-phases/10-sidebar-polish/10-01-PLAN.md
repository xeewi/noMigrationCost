```typescript
export function useActiveSection(ids: string[]): string
```

Implementation details:
- Import `useEffect`, `useRef`, `useState` from 'react'
- State: `const [activeId, setActiveId] = useState<string>(() => ids[0] ?? '')` — lazy initializer sets first heading as active on mount (prevents empty highlight on initial load)
- Ref: `const ratioMap = useRef<Map<string, number>>(new Map())`
- `useEffect` depending on `[ids]`:
  - Guard: `if (ids.length === 0) return;`
  - Create IntersectionObserver with callback that:
    - Iterates `entries`, sets `ratioMap.current.set(entry.target.id, entry.intersectionRatio)` for each
    - Finds the entry with the highest ratio: iterate `ratioMap.current`, track `best` (string) and `bestRatio` (number)
    - If `best` is non-empty AND `bestRatio > 0`, call `setActiveId(best)` — the `> 0` guard prevents clearing active state when all headings scroll out of the observation zone
  - Observer options: `{ rootMargin: '-10% 0px -80% 0px', threshold: [0, 0.1, 0.25, 0.5, 0.75, 1.0] }`
  - Loop over `ids`, call `document.getElementById(id)` and `observer.observe(el)` for each non-null element
  - Return cleanup: `() => observer.disconnect()`
- Return `activeId`
