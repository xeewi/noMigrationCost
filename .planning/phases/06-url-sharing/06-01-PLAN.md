Create `src/lib/url-state.ts` with the following exports:

1. **`ShareableState` interface** — compact keys for URL encoding (per D-01, SHARE-01):
```typescript
export interface ShareableState {
  v: 1;                            // schema version
  th: [number, number, number, number]; // headcounts (Junior/Mid/Senior/Lead)
  tr: [number, number, number, number]; // hourly rates
  sm: 'sp' | 'dh';                 // sizingMode
  sp: number;                      // storyPoints
  vel: number;                     // velocity
  sw: number;                      // sprintWeeks
  dv: number;                      // directValue
  du: 'h' | 'd';                   // directUnit
  hy: number;                      // horizonYears
  gf: number;                      // generalizationFactor
  pf: number;                      // portingFactor
  dr: number;                      // divergenceRate
  mrs: number;                     // maintenanceRateShared
  mrd: number;                     // maintenanceRateDuplicated
  bdf: number;                     // bugDuplicationFactor
  nc: number;                      // nbConsumingCodebases
  tab: 'c' | 's';                  // active tab (comparison/standalone)
}
```

2. **`encodeAppState` function** — accepts all App.tsx state fields as parameters (team: SeniorityRow[], sizingMode: SizingMode, storyPoints: number, velocity: number, sprintWeeks: number, directValue: number, directUnit: DirectHoursUnit, horizonYears: number, advancedParams: AdvancedParamsState, nbConsumingCodebases: number, activeTab: string). Maps `SizingMode` values: `'story-points'` -> `'sp'`, `'direct-hours'` -> `'dh'`. Maps `DirectHoursUnit`: `'hours'` -> `'h'`, `'days'` -> `'d'`. Maps `activeTab`: `'comparison'` -> `'c'`, `'standalone'` -> `'s'`. Extracts headcounts and rates from team array (4 fixed rows in Junior/Mid/Senior/Lead order). Returns Base64URL string: `btoa(JSON.stringify(state)).replace(/\+/g, '-').replace(/\//g, '_').replace(/=+$/, '')`.

3. **`decodeAppState` function** — accepts hash string (without `#`), returns `ShareableState | null`. Restores Base64 padding: `hash.replace(/-/g, '+').replace(/_/g, '/').padEnd(hash.length + (4 - (hash.length % 4)) % 4, '=')`. Wraps `atob` + `JSON.parse` in try/catch, returns `null` on any error (graceful fallback for malformed hashes).

4. **`applyStateToSetters` function** — accepts decoded `ShareableState` and an object of all App.tsx state setters:
```typescript
interface StateSetters {
  setTeam: (t: SeniorityRow[]) => void;
  setSizingMode: (m: SizingMode) => void;
  setStoryPoints: (n: number) => void;
  setVelocity: (n: number) => void;
  setSprintWeeks: (n: number) => void;
  setDirectValue: (n: number) => void;
  setDirectUnit: (u: DirectHoursUnit) => void;
  setHorizonYears: (n: number) => void;
  setAdvancedParams: (p: AdvancedParamsState) => void;
  setNbConsumingCodebases: (n: number) => void;
  setActiveTab: (t: string) => void;
}
```
Reverse-maps short keys to full values: `'sp'` -> `'story-points'`, `'dh'` -> `'direct-hours'`, `'h'` -> `'hours'`, `'d'` -> `'days'`, `'c'` -> `'comparison'`, `'s'` -> `'standalone'`. Reconstructs team array from `th` and `tr` arrays using labels `['Junior', 'Mid', 'Senior', 'Lead']`.

5. **`getDefaultTeam` function** — returns `SeniorityRow[]` with headcount 0 and `SENIORITY_DEFAULTS` rates for each level (Junior/Mid/Senior/Lead). This is the canonical reset target.

6. **`getDefaultAdvancedParams` function** — returns `AdvancedParamsState` with all `ENGINE_DEFAULTS` values. Canonical reset target for advanced params.

Import from `@/engine/types`: `SeniorityRow`, `SeniorityLevel`, `SizingMode`, `DirectHoursUnit`, `SENIORITY_DEFAULTS`, `ENGINE_DEFAULTS`.
Import from `@/components/AdvancedParameters`: `AdvancedParamsState`.
