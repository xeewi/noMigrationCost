    ## calcTeamAvgRate(team: SeniorityRow[]): number
    - Input: [{Junior, 2, 32}, {Mid, 1, 40}, {Senior, 0, 51}, {Lead, 0, 67}] → (2*32 + 1*40) / 3 = 34.67
    - Input: all headcount=0 → returns 0 (no division by zero)
    - Only rows with headcount > 0 are included

    ## calcDevHours(sizing: SizingInputs): number
    - SP mode: storyPoints=20, velocity=10 (SP/sprint), sprintWeeks=2 → sprints=2, hours = 2 * 2 * 35 = 140
    - Direct hours mode: directValue=100, directUnit='hours' → 100
    - Direct days mode: directValue=20, directUnit='days' → 20 * 7 = 140

    - annualMaintenanceCost = 33800 * 0.18 = 6084
    - totalMaintenanceCost = 6084 * 3 = 18252
    - totalStandaloneCost = 33800 + 18252 = 52052
    - breakdown has 2 rows: "Initial Development" and "Annual Maintenance (x3 years)"
    - Each row has hours, cost, percentage (percentage sums to 100)
